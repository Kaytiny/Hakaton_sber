Задание: ИИ-агент для генерации признаков
Вам нужно реализовать ИИ-агента на базе LLM GigaChat-2-Max, который автоматически работает с табличными данными:

обрабатывает входные таблицы с помощью python-библиотек;
понимает их структуру по описанию;
генерирует признаки;
оценивает их полезность на обучающей выборке;
выбирает лучший набор признаков (максимум 5 признаков);
сохраняет сгенерированные признаки для всех сэмплов датасетов 
train
 и 
test
.
Главная цель задания — показать, что ваш агент умеет самостоятельно придумывать и проверять хорошие признаки для модели бустинга.
